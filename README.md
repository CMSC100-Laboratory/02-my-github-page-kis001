# Steps on how to create Github page  
1. Create a Github account  
2. Create a new repository  
3. Upload your HTML, CSS, and asset files to the repository  
4. Enable GitHub Pages in the repository settings  
5. Access your live site in the provided URL

My key takeaways from this exercise consist of handling flexboxes, making content readable, proper usage of GitHub, and reviewing fundamentals of HTML and CSS.
